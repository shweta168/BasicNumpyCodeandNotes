2.Comparison between numpy array and python list.
	-Numpy need to be installed, whereas python list are in built.
	-Less memory is required for numpy arrays, more memory is required for python list.
CODE: 
	import numpy as np
	import sys
	S=1000
	l1=range(S)
	print("Memory",sys.getsizeof(l1))
	print("Whole memory",sys.getsizeof(l1)*len(S))	
	a=np.arange(S)
	print("Memory",a.itemsize)
	print("Whole",a.itemsize*a.size)

	-Less time is required for numpy array as compared to python list.
CODE: 
	import numpy as np

	initial_time=time.time();

12. What is copy and view?
	copy-->new array.
		Any changes in the copy will not affect the original array and visa versa.
		Has its own data.
	View--> copy of original array.
		Any changes made in view wil affect the original array and visa versa.
		Doesnot have its own data.
